O devtube é um site que filtra o conteudo do youtube, mostrando os videos mais populares de uma linguagem de programação

## Depencias usadas 

- React
- React-router-dom
- Axios
- youtube data api v3

## Para rodar o app
- Acesse a pasta pelo terminal
- npm updata
- npm start
- divirta-se!
- Ou clique [aqui](https://devtubebr.herokuapp.com/)
